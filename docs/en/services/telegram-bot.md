# Telegram Bot Service Overview

> Updated: 2026-04-20

Telegram Bot is an independent Dujiao-Next extension that extends your site capabilities into Telegram scenarios. It is suitable for sites that need private-domain reach, message notifications, quick entry points, and operational broadcasts.

> Note: this page describes the standalone Telegram Bot feature, not the Telegram login settings in system configuration.

## 1. Pricing and Service Term

- Price: `598USD` / one-time license
- Service term: one-time license
- Update support: one-time license with ongoing updates, feature iteration, and version upgrades

## 2. Suitable Scenarios

- You want to extend user reach from the web to Telegram
- You need a Bot to handle welcome messages, quick menus, and common entry points
- You need to send campaign notices, marketing messages, and operational broadcasts to Telegram users
- You need a dedicated place to review Bot service connectivity, license status, and configuration sync status

## 3. Currently Supported Features

### 3.1 Basic Bot Settings

- Enable or disable the Telegram Bot
- Configure the Bot display name, description, cover image, and support link
- Configure the default language

### 3.2 Welcome Message

- Configure welcome messages for new users
- Maintain welcome copy by language

### 3.3 Custom Menu

- Configure Bot menu items
- Set menu text, order, action type, and action value
- Suitable for quick entries such as the homepage, help page, support page, and campaign page

### 3.4 Connection Status and Runtime Information

- Review Telegram Bot service connectivity
- Check webhook status, Bot version, configuration version, and last online time
- Review license status, license expiration time, and warning messages

### 3.5 Channel Client Management

- Create and manage Telegram Bot channel clients
- Maintain the Bot Token, callback URL, status switch, and secret reset

### 3.6 Broadcast Messaging

- Review historical broadcast records
- Create new broadcast tasks
- Supported recipients:
  - All Telegram Bot users
  - Selected users
- Selected users can be filtered by:
  - Nickname
  - Telegram username
  - Telegram User ID
  - Time range
- Message content supports Telegram HTML format
- Upload attachments and send them together with the broadcast task
- Track successful sends, failed sends, completion time, and error messages
