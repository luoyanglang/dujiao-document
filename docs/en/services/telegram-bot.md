# Telegram Bot Service Overview

> Updated: 2026-03-11

Telegram Bot is an independent Dujiao-Next extension that brings your storefront workflows into Telegram. It is suitable for sites that need direct user reach, message delivery, shortcut menus, and operational broadcasts.

> Note: this page describes the standalone Telegram Bot feature, not the Telegram login settings in system configuration.

## 1. Pricing and Term

- Price: limited-time early bird `398USD` / lifetime
- License term: lifetime license
- Update policy: continuous updates, feature iteration, and version upgrades are included with the lifetime license

## 2. Best Fit Scenarios

- You want to extend user reach from the web to Telegram
- You need a Bot for welcome messages, shortcut menus, and quick-entry actions
- You want to send campaign notices, operational updates, or marketing broadcasts to Telegram users
- You need a separate place to review Bot connectivity, license status, and configuration sync state

## 3. Currently Available Features

### 3.1 Basic Bot Settings

- Enable or disable the Telegram Bot
- Configure display name, description, cover image, and support URL
- Set the default language

### 3.2 Welcome Message

- Configure welcome messages for new users
- Maintain localized welcome copy by language

### 3.3 Custom Menu

- Configure Bot menu items
- Set menu labels, order, action type, and action value
- Suitable for homepage links, help pages, support links, campaign pages, and other quick entries

### 3.4 Connection and Runtime Status

- Review Telegram Bot service connectivity
- Check webhook status, Bot version, config version, and last seen time
- Review license status, expiration time, and warning messages

### 3.5 Channel Client Management

- Create and manage Telegram Bot channel clients
- Maintain Bot Token, callback URL, status switches, and secret reset

### 3.6 Broadcast Messaging

- Review historical broadcast records
- Create new broadcast tasks
- Recipient types:
  - All Telegram Bot users
  - Selected users
- Selected-user filtering supports:
  - Display name
  - Telegram username
  - Telegram user ID
  - Time range
- Message content supports Telegram HTML formatting
- Attachments can be uploaded and sent with the broadcast
- Track success count, failed count, completion time, and error message
