# Telegram Bot 服务介绍

> 更新时间：2026-03-11

Telegram Bot 是 Dujiao-Next 的独立扩展功能，用于把站点能力延伸到 Telegram 场景，适合需要私域触达、消息通知、快捷入口和运营群发的站点。

> 注意：这里介绍的是独立的 Telegram Bot 功能，不是系统设置中的 Telegram 登录配置。

## 1. 售价与服务周期

- 售价：`398USD` / 年
- 服务周期：按年授权
- 更新支持：授权期内支持持续更新、功能迭代与版本升级

## 2. 适合什么场景

- 希望把用户触达从网页扩展到 Telegram
- 需要用 Bot 承接欢迎消息、快捷菜单和常用入口
- 需要面向 Telegram 用户做活动通知、营销消息和运营群发
- 需要独立查看 Bot 服务连接状态、授权状态和配置同步情况

## 3. 当前支持的功能

### 3.1 Bot 基础设置

- 启用或停用 Telegram Bot
- 配置 Bot 展示名称、描述、封面图和支持链接
- 配置默认语言

### 3.2 欢迎消息

- 支持为新用户配置欢迎消息
- 支持按语言维护欢迎文案

### 3.3 自定义菜单

- 支持配置 Bot 菜单项
- 可设置菜单文案、排序、动作类型和动作值
- 适合放置首页、帮助页、客服页、活动页等快捷入口

### 3.4 连接状态与运行信息

- 查看 Telegram Bot 服务连接状态
- 查看 webhook 状态、Bot 版本、配置版本、最近在线时间
- 查看授权状态、授权到期时间和告警信息

### 3.5 渠道客户端管理

- 支持创建和管理 Telegram Bot 渠道客户端
- 支持维护 Bot Token、回调地址、状态开关和密钥重置

### 3.6 消息群发

- 支持查看历史群发记录
- 支持新建群发任务
- 接收人支持：
  - 全部 Telegram Bot 用户
  - 指定用户
- 指定用户支持按以下条件筛选：
  - 昵称
  - Telegram 用户名
  - Telegram User ID
  - 时间范围
- 消息内容支持 Telegram HTML 格式
- 支持上传附件并随群发任务一起发送
- 支持统计发送成功数、失败数、完成时间和错误信息
