# 分销推广

> 更新时间：2026-03-28

Dujiao-Next 内置分销推广系统，用户可通过推广链接邀请他人购买并获得佣金。系统支持自动归因、佣金计算、确认期机制和提现审核。

---

## 1. 功能概览

| 功能 | 说明 |
|------|------|
| 推广开通 | 用户申请成为推广员，获取专属推广码 |
| 推广链接 | 通过链接或推广码追踪访客来源 |
| 自动归因 | 30 天窗口期内，访客下单自动关联推广员 |
| 佣金计算 | 订单支付后自动生成佣金记录 |
| 确认期 | 佣金经过确认期后才可提现 |
| 提现管理 | 用户申请提现，管理员审核 |
| 推广仪表盘 | 查看点击、订单、佣金等数据 |

---

## 2. 开通推广

### 2.1 用户端

1. 在前台进入「推广中心」
2. 点击「开通推广」
3. 系统自动生成 8 位推广码（大写字母 + 数字）

> 需要管理员在后台开启分销功能后用户才能开通。

### 2.2 管理员配置

后台「设置 → 分销设置」中配置：

| 字段 | 说明 |
|------|------|
| 是否启用 | 分销功能总开关 |
| 佣金比例 | 订单金额的佣金百分比 |
| 确认天数 | 佣金从「待确认」到「可提现」的等待天数（0 = 立即可提现） |
| 最低提现金额 | 申请提现的最低门槛 |
| 提现渠道 | 支持的提现方式列表 |

---

## 3. 推广链接

推广员的专属链接格式：

```
https://你的域名/?aff=推广码
```

例如推广码为 `AB3CD5EF`，则推广链接为：

```
https://你的域名/?aff=AB3CD5EF
```

用户也可以手动分享推广码，其他用户在访问站点时带上 `?aff=推广码` 参数即可。

---

## 4. 访客归因

### 4.1 归因规则

系统通过以下方式关联推广员与下单用户：

1. **浏览器指纹匹配**：访客点击推广链接后，30 天内在同一浏览器下单会自动归因
2. **推广码匹配**：通过 URL 中的 `aff` 参数直接归因

> 优先使用浏览器指纹匹配（取最近一次有效点击），其次使用推广码匹配。

### 4.2 去重机制

- 同一访客在 10 分钟内对相同页面的重复点击不会记录
- 用户不能成为自己的推广员（自推无效）

---

## 5. 佣金计算

### 5.1 计算规则

订单支付成功后自动生成佣金：

```
佣金基数 = Σ（商品总价 - 优惠券折扣）  // 仅计算开启了分销的商品
佣金金额 = 佣金基数 × 佣金比例 / 100
```

- 只有开启了「允许分销」的商品才参与佣金计算
- 佣金按订单级别生成，每个订单最多一条佣金记录
- 同一订单不会重复生成佣金

### 5.2 佣金状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 待确认 | `pending_confirm` | 等待确认期结束 |
| 可提现 | `available` | 确认期已过，可申请提现 |
| 已提现 | `withdrawn` | 已包含在提现请求中 |
| 已拒绝 | `rejected` | 因订单取消/退款等原因无效 |

### 5.3 状态流转

```
订单支付 → 待确认
            ↓ （确认期到期）
          可提现
            ↓ （用户申请提现）
          已提现

订单取消/退款 → 已拒绝（跳过已在提现流程中的佣金）
```

---

## 6. 提现流程

### 6.1 用户申请

1. 在「推广中心」查看可提现余额
2. 点击「申请提现」
3. 输入提现金额（需 ≥ 最低提现金额）
4. 选择提现渠道并填写收款信息
5. 提交后等待管理员审核

> 申请时系统会将对应金额的佣金锁定到提现请求中。如果最后一笔佣金金额大于所需金额，系统会自动拆分。

### 6.2 管理员审核

后台「分销管理 → 提现管理」中处理：

- **通过**：标记佣金为「已提现」，管理员线下完成打款
- **拒绝**：填写拒绝原因，锁定的佣金释放回「可提现」状态

---

## 7. 推广仪表盘

用户在前台「推广中心」可查看：

| 数据 | 说明 |
|------|------|
| 推广码 | 个人专属推广码 |
| 推广链接 | 完整的推广 URL |
| 总点击数 | 推广链接被点击的次数 |
| 有效订单数 | 通过推广产生的已完成订单数 |
| 转化率 | 有效订单数 / 总点击数 × 100% |
| 待确认佣金 | 尚在确认期内的佣金总额 |
| 可提现佣金 | 已确认可申请提现的佣金总额 |
| 已提现佣金 | 历史已提现的佣金总额 |

---

## 8. 订单退款与佣金

- **订单取消**：关联的佣金自动标记为「已拒绝」
- **部分退款**：按退款比例等比扣减佣金
  - 退款比例 = 退款金额 / 剩余可退金额
  - 佣金基数和佣金金额按比例减少
  - 如果佣金减至 0 或以下，自动标记为「已拒绝」
- 已进入提现流程的佣金不受订单取消影响

---

## 9. 注意事项

- 推广功能需管理员开启后用户才能使用
- 推广码为 8 位大写字母和数字组合（排除了易混淆的 I、O）
- 归因窗口期为 30 天，超过后点击记录不再用于归因
- 佣金基数扣除了优惠券折扣，但不扣除钱包支付部分
- 商品需在后台单独开启「允许分销」开关
