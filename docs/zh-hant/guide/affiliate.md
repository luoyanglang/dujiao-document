# 分銷推廣

> 更新時間：2026-03-28

Dujiao-Next 內建分銷推廣系統，使用者可透過推廣連結邀請他人購買並獲得佣金。系統支援自動歸因、佣金計算、確認期機制和提現審核。

---

## 1. 功能概覽

| 功能 | 說明 |
|------|------|
| 推廣開通 | 使用者申請成為推廣員，取得專屬推廣碼 |
| 推廣連結 | 透過連結或推廣碼追蹤訪客來源 |
| 自動歸因 | 30 天窗口期內，訪客下單自動關聯推廣員 |
| 佣金計算 | 訂單付款後自動產生佣金紀錄 |
| 確認期 | 佣金經過確認期後才可提現 |
| 提現管理 | 使用者申請提現，管理員審核 |
| 推廣儀表板 | 查看點擊、訂單、佣金等數據 |

---

## 2. 開通推廣

### 2.1 使用者端

1. 在前台進入「推廣中心」
2. 點擊「開通推廣」
3. 系統自動產生 8 位推廣碼（大寫字母 + 數字）

> 需要管理員在後台開啟分銷功能後使用者才能開通。

### 2.2 管理員設定

後台「設定 → 分銷設定」中設定：

| 欄位 | 說明 |
|------|------|
| 是否啟用 | 分銷功能總開關 |
| 佣金比例 | 訂單金額的佣金百分比 |
| 確認天數 | 佣金從「待確認」到「可提現」的等待天數（0 = 立即可提現） |
| 最低提現金額 | 申請提現的最低門檻 |
| 提現管道 | 支援的提現方式列表 |

---

## 3. 推廣連結

推廣員的專屬連結格式：

```
https://你的網域/?aff=推廣碼
```

例如推廣碼為 `AB3CD5EF`，則推廣連結為：

```
https://你的網域/?aff=AB3CD5EF
```

使用者也可以手動分享推廣碼，其他使用者在造訪網站時帶上 `?aff=推廣碼` 參數即可。

---

## 4. 訪客歸因

### 4.1 歸因規則

系統透過以下方式關聯推廣員與下單使用者：

1. **瀏覽器指紋比對**：訪客點擊推廣連結後，30 天內在同一瀏覽器下單會自動歸因
2. **推廣碼比對**：透過 URL 中的 `aff` 參數直接歸因

> 優先使用瀏覽器指紋比對（取最近一次有效點擊），其次使用推廣碼比對。

### 4.2 去重機制

- 同一訪客在 10 分鐘內對相同頁面的重複點擊不會記錄
- 使用者不能成為自己的推廣員（自推無效）

---

## 5. 佣金計算

### 5.1 計算規則

訂單付款成功後自動產生佣金：

```
佣金基數 = Σ（商品總價 - 優惠券折扣）  // 僅計算開啟了分銷的商品
佣金金額 = 佣金基數 × 佣金比例 / 100
```

- 只有開啟了「允許分銷」的商品才參與佣金計算
- 佣金按訂單級別產生，每個訂單最多一條佣金紀錄
- 同一訂單不會重複產生佣金

### 5.2 佣金狀態

| 狀態 | 標識 | 說明 |
|------|------|------|
| 待確認 | `pending_confirm` | 等待確認期結束 |
| 可提現 | `available` | 確認期已過，可申請提現 |
| 已提現 | `withdrawn` | 已包含在提現請求中 |
| 已拒絕 | `rejected` | 因訂單取消/退款等原因無效 |

### 5.3 狀態流轉

```
訂單付款 → 待確認
            ↓ （確認期到期）
          可提現
            ↓ （使用者申請提現）
          已提現

訂單取消/退款 → 已拒絕（跳過已在提現流程中的佣金）
```

---

## 6. 提現流程

### 6.1 使用者申請

1. 在「推廣中心」查看可提現餘額
2. 點擊「申請提現」
3. 輸入提現金額（需 ≥ 最低提現金額）
4. 選擇提現管道並填寫收款資訊
5. 提交後等待管理員審核

> 申請時系統會將對應金額的佣金鎖定到提現請求中。如果最後一筆佣金金額大於所需金額，系統會自動拆分。

### 6.2 管理員審核

後台「分銷管理 → 提現管理」中處理：

- **通過**：標記佣金為「已提現」，管理員線下完成打款
- **拒絕**：填寫拒絕原因，鎖定的佣金釋放回「可提現」狀態

---

## 7. 推廣儀表板

使用者在前台「推廣中心」可查看：

| 數據 | 說明 |
|------|------|
| 推廣碼 | 個人專屬推廣碼 |
| 推廣連結 | 完整的推廣 URL |
| 總點擊數 | 推廣連結被點擊的次數 |
| 有效訂單數 | 透過推廣產生的已完成訂單數 |
| 轉換率 | 有效訂單數 / 總點擊數 × 100% |
| 待確認佣金 | 尚在確認期內的佣金總額 |
| 可提現佣金 | 已確認可申請提現的佣金總額 |
| 已提現佣金 | 歷史已提現的佣金總額 |

---

## 8. 訂單退款與佣金

- **訂單取消**：關聯的佣金自動標記為「已拒絕」
- **部分退款**：按退款比例等比扣減佣金
  - 退款比例 = 退款金額 / 剩餘可退金額
  - 佣金基數和佣金金額按比例減少
  - 如果佣金減至 0 或以下，自動標記為「已拒絕」
- 已進入提現流程的佣金不受訂單取消影響

---

## 9. 注意事項

- 推廣功能需管理員開啟後使用者才能使用
- 推廣碼為 8 位大寫字母和數字組合（排除了易混淆的 I、O）
- 歸因窗口期為 30 天，超過後點擊紀錄不再用於歸因
- 佣金基數扣除了優惠券折扣，但不扣除錢包付款部分
- 商品需在後台單獨開啟「允許分銷」開關
