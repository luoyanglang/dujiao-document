# Telegram Bot 服務介紹

> 更新時間：2026-03-11

Telegram Bot 是 Dujiao-Next 的獨立擴充功能，用於把站點能力延伸到 Telegram 場景，適合需要私域觸達、訊息通知、快捷入口與營運群發的站點。

> 注意：這裡介紹的是獨立的 Telegram Bot 功能，不是系統設定中的 Telegram 登錄配置。

## 1. 售價與服務週期

- 售價：限時早鳥價 `398USD` / 終身
- 服務週期：終身授權
- 更新支援：終身授權，支援持續更新、功能迭代與版本升級

## 2. 適合什麼場景

- 希望把使用者觸達從網頁延伸到 Telegram
- 需要用 Bot 承接歡迎訊息、快捷選單與常用入口
- 需要面向 Telegram 使用者做活動通知、行銷訊息與營運群發
- 需要獨立查看 Bot 服務連接狀態、授權狀態與設定同步情況

## 3. 目前支援的功能

### 3.1 Bot 基礎設定

- 啟用或停用 Telegram Bot
- 配置 Bot 顯示名稱、描述、封面圖與支援連結
- 配置預設語言

### 3.2 歡迎訊息

- 支援為新使用者配置歡迎訊息
- 支援按語言維護歡迎文案

### 3.3 自訂選單

- 支援配置 Bot 選單項
- 可設定選單文案、排序、動作類型與動作值
- 適合放置首頁、幫助頁、客服頁、活動頁等快捷入口

### 3.4 連接狀態與執行資訊

- 查看 Telegram Bot 服務連接狀態
- 查看 webhook 狀態、Bot 版本、設定版本、最近在線時間
- 查看授權狀態、授權到期時間與告警資訊

### 3.5 渠道客戶端管理

- 支援建立與管理 Telegram Bot 渠道客戶端
- 支援維護 Bot Token、回呼地址、狀態開關與密鑰重置

### 3.6 訊息群發

- 支援查看歷史群發記錄
- 支援建立新的群發任務
- 接收人支援：
  - 全部 Telegram Bot 使用者
  - 指定使用者
- 指定使用者支援以下篩選條件：
  - 暱稱
  - Telegram 使用者名稱
  - Telegram User ID
  - 時間範圍
- 訊息內容支援 Telegram HTML 格式
- 支援上傳附件並隨群發任務一起發送
- 支援統計發送成功數、失敗數、完成時間與錯誤資訊
