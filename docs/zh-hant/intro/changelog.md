# 更新日誌

## [v1.0.4] - 2026-04-22

### 更新內容：
- 優化郵件通知相容性，減少部分信箱情境下的收發異常，站點通知更穩定。
- 修復 SQLite 環境下後台手動退款可能出現的流程卡頓問題，退款處理更可靠。
- 優化前台多處頁面與通用元件，商品瀏覽、下單與登入等常用流程更順暢。
- 補充商品交付使用說明文件，適合付款後需要展示額外指引的交付情境。
- 優化 Docker Compose 部署文件提示，首次部署與排查設定問題更省心。
